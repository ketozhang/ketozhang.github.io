The classical Keplarian set of parameters that describe the state of the satellite is given by:

  1. \(a\) : Semimajor Axis
  2. \(e\) : eccentricity
  3. \(i\) : inclination
  4. \(\Omega\) : longitude of acending node
  5. \(\omega\) : argument of perigee
  6. \(\nu\) : True anomaly
